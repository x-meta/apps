# Action
动作(Action)相当于函数和方法，可以用来编写程序。

### 1.安装应用
执行setup.cmd或setup.sh安装应用。

### 2.执行Action模型
通过run.cmd或run.sh执行Action模型。

### 3.运行方式
动作模型的属性mode用于设置运行方式，mode=edit是编辑方式，mode=run是运行方式。

初始的要运行的动作模型是Action.dml，默认的运行方式是edit。

#### 3.1.编辑方式
即执行后打开编辑器编辑动作模型自己。
